Assignment 07 - Arrays and Looping

3 questions for this assignment:

  1. Make an Array of strings and assign it to a variable. You may make the strings a list of anything you like and use a variable name of your choice.
  2. Use one of the methods of the Array object to insert a new element to the beginning of your array.
  3. Using the Array you made in the previous steps, iterate over its elements and write each one to the console.
